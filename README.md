# Career Guidance Chatbot

## Description
A chatbot built using Google Dialogflow to provide career guidance for students and freshers.

## Features
- Welcome
- Resume Tips
- Resume Format
- Resume Objective
- Resume Skills
- Resume Projects
- Interview Preparation
- HR Interview
- Technical Interview
- Career Roadmap
- Skill Suggestions
- Job Search Tips
- Course Recommendations
- Contact Support
- End Chat
## Technology Used
- Google Dialogflow ES
